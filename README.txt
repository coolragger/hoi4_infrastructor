 Hearts of Iron IV: Infrastructor
**********************************
Here is the instructions for using this Powershell script.

1. Start the Hearts of Iron IV Paradox launcher.

2. Click to the mod section in the launcher and click on the mod tool button.

3. Create your mod and close down Hearts of Iron IV.

4. Use the explorer and locate you new mod root folder. the default is C:\Users\<"Your Username">\Documnets\Paradox Interactive\Hearts of Iron IV\mod

5. Copy and paste the hoi4-mod-setup.ps1 file in the root folder and run the script.

Enjoy!
